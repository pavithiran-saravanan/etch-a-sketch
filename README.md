# Etch-a-sketch
Web version of a popular kids game Etch-a-sketch. 

# Demo
Check out the live demo of the project here https://pavithiran-saravanan.github.io/etch-a-sketch/

# Built With
- HTML
- CSS
- Javascript

# Features
- Resizeable grid
- A fun RGB sketch mode
- Mobile responsive
